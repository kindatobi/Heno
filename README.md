# Heno

A modern, full-stack fashion e-commerce web application built with Next.js 15, featuring a polished user experience with smooth animations, secure authentication, and seamless payment processing.

## Features

- **Modern UI/UX** - Built with Radix UI components and Tailwind CSS for a sleek, accessible interface
- **Smooth Animations** - GSAP and Lenis integration for fluid page transitions and scroll effects
- **Drag & Drop** - Product management with @dnd-kit for intuitive admin controls
- **Secure Authentication** - Powered by Better Auth with PostgreSQL session management
- **Payment Processing** - Stripe integration for secure checkout and transactions
- **File Uploads** - UploadThing integration for product images and media
- **Email Notifications** - Resend integration for transactional emails
- **Background Jobs** - Inngest for reliable task processing and workflows
- **Form Management** - React Hook Form with Zod validation for type-safe forms
- **State Management** - Zustand for lightweight, scalable state handling
- **Database** - Prisma ORM with PostgreSQL and connection pooling via @upstash/redis
- **Data Visualization** - Recharts for admin analytics and reporting


## Tech Stack

- **Framework:** Next.js 15 (React 19)
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Database:** PostgreSQL with Prisma ORM
- **Authentication:** Better Auth
- **Payments:** Stripe
- **Animations:** GSAP, Lenis
- **UI Components:** Radix UI, shadcn/ui patterns
- **Icons:** Phosphor Icons, Lucide React, Tabler Icons
